# Indexation Google manuelle

Liste prioritaire pour demander l'indexation dans Google Search Console.

## Methode

1. Ouvrir Google Search Console.
2. Selectionner la propriete `dsdepannage.fr`.
3. Aller dans `Inspection de l'URL`.
4. Copier une URL depuis `urls-prioritaires.txt`.
5. Attendre le test Google.
6. Cliquer sur `Demander une indexation`.
7. Noter dans `suivi-indexation.csv` la date et le statut.

## Ordre conseille

1. Accueil.
2. Page tarifs.
3. Pages `serrurier Paris` des 20 arrondissements.
4. Pages `ouverture porte claquee` des 20 arrondissements.

Ne pas forcer 300 pages le meme jour si Search Console bloque ou ralentit.
